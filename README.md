# LineageOS Port (For Samsung Galaxy A05s)

This repository contains an automated **LineageOS porting pipeline** powered by **GitHub Actions**.

The goal of this project is to:
- Take a **LineageOS GSI**
- Inspect, adapt, and prepare it for a **Samsung Galaxy A05s**
- Automate the entire process using **GitHub Actions**

This repo is currently in **early stages**
**IMPORTANT-**
for now builds would be made for **A075F varients only!**

Stay tuned!
